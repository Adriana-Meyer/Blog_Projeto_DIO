HTML: Linguagem de Marcação de Hipertexto

O HTML (Linguagem de Marcação de HiperTexto) é o bloco de construção mais básico da web. Ele é responsável por definir o significado e a estrutura do conteúdo de uma página na web. Fazendo uma analogia, o HTML seria como o esqueleto de um corpo.

O termo "Hipertexto" refere-se aos links que conectam páginas entre si, seja dentro de um único site ou entre sites.

O HTML usa "Marcação" através de tags específicas para estruturar o texto, imagem e outros conteúdos para exibição em um navegador da Web. Para conhecer mais detalhes sobre cada uma das tags disponíveis, acesse o site da W3 Schools.

CSS: Cascading Style Sheets

O CSS (Cascading Style Sheets ou Folhas de Estilo em Cascata) é uma linguagem de estilo usada para descrever o visual de um documento escrito em HTML ou em XML.

O CSS descreve como elementos serão exibidos na tela, utilizando seletores de classes ou Ids para definir estilos como cores dos elementos, tipo e tamanho das fontes, sombras, opacidade e transparência de imagens, entre outros aspectos. Fazendo uma analogia, seria como a pele em um corpo.

O CSS, atualmente na versão CSS3, é uma das principais linguagens da web e é padronizada em navegadores de acordo com diretrizes da W3C. Para saber mais detalhes sobre os atributos, acesse a W3 Schools.

JavaScript

O JavaScript (também conhecido como JS) é uma linguagem de programação interpretada estruturada, de script em alto nível com tipagem dinâmica fraca e multiparadigma (protótipos, orientado a objeto, imperativo e funcional).

Juntamente com HTML e CSS, o JavaScript é uma das três principais tecnologias da World Wide Web. Fazendo uma analogia, seria como os músculos em um corpo. JavaScript permite a criação de páginas interativas e, portanto, é uma parte essencial dos aplicativos da web. A grande maioria dos sites a utiliza, e todos os principais navegadores têm um mecanismo JavaScript dedicado para executá-lo.

É atualmente a principal linguagem para programação client-side em navegadores web. É também bastante utilizada do lado do servidor através de ambientes como o node.js.
